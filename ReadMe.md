# 🧠 MindMirror: Reframe Your Perspective

## 🚀 Live Project

🔗 https://github.com/yuvee007/MindMirror

---

## 📌 Problem Statement

In today’s digital world, people often post thoughts impulsively, without reflecting on tone, bias, or impact. This leads to misunderstandings, unintentional harm, and negative online interactions.

Most platforms attempt to moderate content **after it is posted**, which is reactive and often ineffective.

---

## 💡 Solution

**MindMirror** is a reflection-first communication tool that helps users:

> ✨ *Pause, reflect, and improve their thoughts before sharing them.*

Instead of judging users, the app gently guides them toward more thoughtful, respectful, and balanced expression.

---

## 🎯 Key Features

### 🧠 1. Bias Analyzer (Core Feature)

* Analyzes user input for:

  * Generalizations (e.g., “all”, “always”, “never”)
  * Harsh or aggressive tone
* Provides:

  * Type of issue detected
  * Simple explanation
  * Suggested improvement

---

### ✨ 2. Text Refinement

* “Improve my text” feature
* Rewrites input into a more neutral and constructive version
* Displays:

  * Before vs After comparison

---

### 📊 3. Dashboard

* Stores user history
* View past analyses and refinements
* Delete entries
* Persistent data using backend

---

### 🌍 4. Community Feed

* Users can post refined thoughts
* Anonymous posting option
* Like/support posts
* Tagged content (e.g., tone, generalization)

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/yuvee007/MindMirror.git
cd MindMirror
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variables

Create a `.env` file:

```
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

---

### 4. Run the app

```bash
npm run dev
```
